Simon says game

variables
-------------
Create a panel with a display for level and light buttons

create start game button

Create an array containing the available colors

Create an empty array to store the computer's sequence

Create an empty array to store the player's sequence

Set the starting level to 1

Set the game as not started

create restart button

-------------

functions
-------------
repeat while the game has not ended

add a random color to the computer sequence

display the full computer sequence to the player to memorize in order

clear the player sequence to display next level

after each click compare the player's input with the computer's sequence

if clicked incorrect color display a game over message and restart button

if the player clicks the entire sequence correctly, increase the level and increase the sequence length

-------------

User Stories
-------------

As a user I want to press start to begin the game

As a user I want to see the colors pattern so I can memorize it

As a user I want to click the panels or colors in the same order

As a user I want to know which level I am on and what level I reached

As a user I want to see a game over message if I make a mistake

As a user I want to restart after losing the game
